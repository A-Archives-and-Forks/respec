Een informatieve paragraaf

Als binnen een normatief document een niet-normatieve / alleen informatieve paragraaf moet staan, gebruik dan de CSS class informative op het section element. Dus: (<section id="mijnid" class="informative">). Er komt dan autmatisch onder de titel te staan Dit onderdeel is niet normatief.
